# Wordle Game

_A game that tests your memory and patience and how critical yoy can think with sound effects._

![Screenshot of my game](./assets/wordle-game.png)

## Getting Started

### Play the Game

[Deployed Game](https://github.com/Hussain-Almutawa7/wordle-game)

### How to Play
1. Press the play button after reading the instructions
2. Guess the word of 5 in 6 attempts
3. Press the reset game if you want to play with another word
4. Press how to play to go back to the instructions

### Installation
No instalation is reuired, simply clone the repo to your machine and open `index.html` file in your browser.

```bash
git clone
cd WORDLE-GAME
open index.html
```
